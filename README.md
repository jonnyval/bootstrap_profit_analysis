# Определение наиболее выгодного региона нефтедобычи (ГлавРосГосНефть)

## Описание проекта
В рамках данного проекта для нефтедобывающей компании «ГлавРосГосНефть» разработана модель машинного обучения, которая поможет определить регион, где добыча принесет наибольшую прибыль. Проведен анализ возможных прибыли и рисков с использованием техники Bootstrap.

## Бизнес-задача
* Построить модель для предсказания объема запасов в новых скважинах.
* Выбрать регион с максимальной суммарной прибылью отобранных скважин.
* Оценить риски убытков (вероятность убытков не должна превышать **2.5%**).

## Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas, NumPy, Scikit-learn, SciPy
* **Методы:** Linear Regression, Bootstrap, Доверительные интервалы

## Краткие результаты
Были исследованы данные геологоразведки трех регионов. С помощью процедуры Bootstrap на 1000 подвыборках были рассчитаны распределения прибыли:
1. **Регион 1:** (Характеристики не прошли порог по рискам)
2. **Регион 2:** * Средняя ожидаемая прибыль: ~518.2 млн руб.
   * 95% доверительный интервал: от 108.1 млн до 817.3 млн руб.
   * **Риск убытков: 1.30%**
3. **Регион 3:**
   * Средняя ожидаемая прибыль: ~382.5 млн руб.
   * Риск убытков: 8.20%

**Итоговый вывод:** По результатам моделирования для дальнейшей разработки аналитическим путем предложен **Регион 2**, так как он единственный удовлетворяет строгому условию бизнеса (риск убытков < 2.5%) и при этом показывает высокую среднюю прибыль.
